Used to get the average value in a chunk on the border pixels, this chunk is alligned with the size of the pixel block used on the iteration and changes are saved on the lastIteration matrix since this matrix is the one used to calculate the new values in the next iteration.

Template Parameters
T: Data type
Alloc: Allocator used for row allignment in the matrix values
Parameters

isIsolated: Vector describing if a border is isolated. Convention used: {top; bot; left; right}
rowIndexes: Array of positions in which the matrix rows are grouped
columnIndexes: Array of positions in which the matrix columns are grouped
sizes: Size of the arrays.
