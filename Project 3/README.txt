Para hacer build o ejecutar ejemplos ir a la carpeta scripts

> cd scripts/

verificar que el archivo tenga permisos de ejecucion 

>chmod +x "archivo"

hacer build

> ./build.sh

Los ejemplos vuelven a llamar build

Para ejecutar los ejemplos

> ./ejemploX.sh

donde X : 1,2,etc

Para ejecutar el programa con parametros deseados:

ir a la carpeta donde se encuentra el ejecutable
> cd build/bin

> ./placa --help
